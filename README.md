# Helpdesk Ticketing System

A full-stack helpdesk application that allows users to create, manage, and track support tickets.

## 🚀 Live Demo

Frontend: https://your-netlify-link.netlify.app
Backend API: https://helpdesk-backend-vvbv.onrender.com

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Database: MongoDB Atlas
* Deployment: Netlify (Frontend), Render (Backend)

## ✨ Features

* Create support tickets
* Delete tickets
* Ticket prioritization (AI-based logic)
* Dashboard analytics
* Admin login system

## 📦 API Endpoints

* GET /api/tickets
* POST /api/tickets
* DELETE /api/tickets/:id
